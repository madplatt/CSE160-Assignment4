Notes to Grader: Click canvas for mouse capture/control, uses MDN pointer lock API, may not function on Safari.
Point light is small yellow cube, orbiting above maze. Cheats enabled by default for vertical movement. No spotlight.
Diffuse Red/Blue/Green sliders adjust the color of diffuse light, specular is always white. Debug Textures show normals.
You start facing in the +x direction, move light along X/Y/Z axis w/ relevant sliders.

Try to find the hidden diamonds inside the maze ruins! Toggle on cheats to enable vertical movement and
block breaking/placement within the maze. Click to break/place blocks with cheats enabled.

Please use the browser supporting "canvas".
